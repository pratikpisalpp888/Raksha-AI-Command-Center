# 🏆 Raksha AI - Hackathon Submission Guide

## 📝 Project Description
**Raksha AI** is a state-of-the-art Emergency Command Center designed for women's safety. It integrates Neural AI Voice Synthesis, Real-Time Tactical Mapping, and automated case management to reduce emergency response times.

## 🚀 Key Features to Highlight
1. **Cinematic Command Interface:** High-end Sci-Fi UI for situational awareness.
2. **HD Neural Voice Engine:** Uses advanced TTS to simulate emotional distress calls in Hindi/English.
3. **Tactical Map Tracking:** Real-time "Fly-To" mapping that tracks incoming distress signals instantly.
4. **Interactive Dialogue Tree:** Dynamic AI-human interaction for emergency assessment.

## 🛠️ Setup Instructions (For Judges)
### 1. Backend (Python/FastAPI)
```bash
cd raksha-ai/backend
pip install -r requirements.txt
python main.py
```
### 2. Frontend (React/Vite)
```bash
cd raksha-frontend
npm install
npm run dev
```

## 🎥 Demo Walkthrough Script
1. **Login:** Demonstrate the secure entry and cinematic transition.
2. **Alert:** Run `python simulate_emergency.py`. Explain how the map auto-tracks the caller.
3. **Response:** Handle the live call. Emphasize the **HD Neural Voice** clarity.
4. **Resolve:** Show how cases are archived in the **Resolved Tab** for historical data analysis.
